- 链表
  有点像数组 可以存一堆的数据 
  数组是连续空间，
  链表是不连续的， 更优  有效通过指针联系起来，
  每个节点，LinkedNode => LinkedList
  {val: 1, next: null}

1. 初始化链表 new LinkedNode(2);
   val  next 用指针 可以不连续存储 
2. 遍历链表
   while(node) {
       node = node.next;
   }
3. 数组转变成为链表
    - 第一次循环 val
    - 第二次是next -1
